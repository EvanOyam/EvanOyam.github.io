Vue 实例方法 & Vue use

`Vue use()` 用于使用 Vue 插件，本质是调用 Vue 插件中的 install 方法。Vue 插件本身是一个对象，这个对象中必须包含 install 方法，install 是一个函数。 本质上 `Vue.use()` 就是调用插件中定义的 install 方法，需要在 `new Vue()` 之前调用。
